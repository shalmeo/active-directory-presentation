Домменные службы Active Directory

Домменные службы Active Directory - служба каталогов разработанная компанией Microsoft.

Домменные службы Active Directory используется для централизованного управления IT-инфрастуркурной организации. Например, централизованное управление администрирования учетных записей и персональных компьютеров пользователей.

Структура Active Directory

Служба каталогов Active Directory имеет иерархическую стурктуру состоящую из следующих компонентов:

  • Домен
  • Контроллер домена
  • Лес доменов
  • Дерево доменов
